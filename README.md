# 🤖 Agentic AI Experiments

A collection of practice implementations, experiments, and learning projects focused on **Agentic AI**, exploring frameworks like **LangChain** and **LangGraph**.

This repository contains small prototypes, experiments, and proof-of-concepts created while learning how AI agents work, how workflows can be orchestrated, and how LLMs can be connected with tools, memory, and external systems.

---

## 🛠️ Technologies Used

- Python
- LangChain
- LangGraph
- Large Language Models (LLMs)
- Vector Databases
- Embeddings
- Retrieval-Augmented Generation (RAG)
- APIs and external tools

---

## ⚙️ Setup

Clone the repository:

```bash
git clone https://github.com/Vijay-Junoon/agenticAI-experiments.git
```

Navigate into the project:

```bash
cd agenticAI-experiments
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

**Windows**
```bash
venv\Scripts\activate
```

**Linux/Mac**
```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file and add required API keys:

```env
GROQ_API_KEY=your_key_here
```

*(Required variables depend on the specific experiment.)*

## 🧠 Purpose

The goal of this repository is to explore:

- How LLM applications are built beyond simple prompting
- How autonomous AI agents can plan and execute tasks
- How workflows can be represented as graphs
- How tools, memory, and reasoning can be combined
- Practical implementation patterns for modern AI systems
